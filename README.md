<h1>Gesture Controller</h1>
<p>Простое приложение для управления компьютером с помощью жестов. Камера отслеживает руку и выполняет назначенные пользователем действия на определенный жест: хоткеи, запуск сайта, запуск приложения, кнопки мыши.
Для каждого жеста пользователь назначает действия сам.
  <h2>Функционал:</h2>
  <b>S</b> - При нажатии кнопки считывает текущий жест с камеры и открывает менюшку для ввода названия и действия жеста. Пока что на каждый жест можно назначить только одно действие, но в будущих обновлениях добавлю последовательность. <br>
  <b>L</b> - Открыть список всех жестов. Позволяет либо удалить определенный жест, либо изменить его функционал. <br>
  <b>O</b> - Изменение сенсы мышки для действия special:toggle_follow. <br>
  <b>Q</b> - Выход <br>

  <hr>
  При запуске с аргмуентом --no-preview не показывает окошко с предпросмотром, а также забирает возможность попадать в менюшки настроек.
